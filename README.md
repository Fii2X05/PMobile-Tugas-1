Pertemuan pertama membahas tentang dasar-dasar pengembangan aplikasi di perangkat mobile, menjelaskan juga tentang SDK (Software Development Kit) yang dibutuhkan untuk
mengembangkan aplikasi di perangkat mobile. 

Aplikasi pengembangannya sendiri yang dibutuhkan yaitu:
- Android Studio SDK
- Dart SDK
- Flutter SDK (Sebenarnya flutter sudah termasuk dengan dart SDK, karena flutter sendiri adalah framework untuk dart)
